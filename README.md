# Coordinate Converter: DMS to Decimal Degrees
This is a lightweight Python tool designed to convert geographic coordinates from Degrees, Minutes, and Seconds (DMS) format into Decimal Degrees (DD). 

## About the Project 
This project was born out of my journey to bridge Geodetic Engineering principles with Python programming. It serves as a practical application of my progress in relearning Python fundamentals—moving from basic syntax to functional tools that solve real-world spatial data problems.

## Feature
- Converts Latitude and Longitude from DMS to DD.
- Handles input via a simple Command Line Interface (CLI).
- Includes a recursive loop to allow for multiple calculations in one session.

## How To Use
1. Run the script : python DD_LatLong.py 
2. Enter your name and follow the prompts.
3. Input Degrees, Minutes, and Seconds when requested.
4. Get the precise decimal output for use in GIS software like QGIS or ArcGIS.
