##Overview
During college, I had the chance to learn Python and use it for automated processing, such as coordinate transformations. But after that, I didn’t really use it much. Lately, I’ve been wanting to level up my programming skills, so I decided to dive back into Python. During this upskilling process, I realized my fundamental knowledge was lacking, so I decided to relearn Python from the ground up (with the help of Harvard’s C50 course on Introduction to Programming with Python). A simple program idea that came to mind is a tool for converting latitude and longitude coordinates from degree-minute-second format to decimal degree format.
